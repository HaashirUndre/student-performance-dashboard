# Student Performance Dashboard

A web app built with Python and Streamlit to analyze student performance data.

## Features
- Upload student marks CSV
- Auto-calculate totals, percentages and grades
- Identify the topper
- Subject-wise average visualization
- Grade distribution pie chart

## Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib

## How to Run
pip install streamlit pandas matplotlib
streamlit run student_analyzer.py
